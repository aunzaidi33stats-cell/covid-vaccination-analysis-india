# COVID-19 Vaccination Analysis in India

## Project Overview
This project analyzes India's COVID-19 vaccination campaign using state-wise vaccination time-series data.

## Objectives
- Data Cleaning and Preprocessing
- State-wise Vaccination Analysis
- Vaccination Coverage Comparison
- Time Series Trend Analysis
- Data Visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Plotly
- Streamlit

## Key Findings
- Significant variation exists in vaccination coverage across states.
- Several Union Territories achieved very high vaccination coverage.
- Vaccination rates peaked during major government vaccination drives.
- Vaccination activity gradually declined after 2022.
